Scripts and Functions

Scripts:

Definition: A script is a file containing a series of MATLAB commands. Scripts run in the current workspace and can use and modify existing variables.

Creating and Running Scripts: To create a script, write the commands in a new file with a `.m` extension. Run the script by typing its name (without the `.m` extension) in the Command Window.

Example Script:

Create a file named area_of_circle.m:

radius = input("Please enter the radius: ");
area = pi * radius^2;
disp(['The area of the circle is ', num2str(area)]);

üí° Tip: Use descriptive names for your scripts to easily remember what they do!

Functions:

Definition: Functions are reusable pieces of code that operate on inputs and produce outputs. Functions have their own workspace, separate from the base workspace.

Components: Name, input arguments, and output arguments.

Creating Functions: Define a function in a new file with the same name as the function and a `.m` extension. Use the function keyword to define a function.

Syntax:

function output = functionName(input)
    % Function body
    output = ...;
end

Example Function:

Create a file named calculate_area.m:

function area = calculate_area(radius)
    area = pi * radius^2;
end

Execution:

result = calculate_area(5);

‚ö†Ô∏è Gotcha: Remember, functions have their own workspace! Variables created within a function do not affect the base workspace unless explicitly returned.

Control Flow and Conditional Statements ‚öôÔ∏è

Control flow:

Statements allow you to execute certain sections of code based on conditions or repeat them a specified number of times. This includes conditional statements and loops.

Conditional Statements:

Conditional statements execute different code blocks based on whether a condition is true or false.

Syntax:

if condition
    % Code to execute if condition is true
elseif another_condition
    % Code to execute if another_condition is true
else
    % Code to execute if none of the above are true
end

Example:

num = input('Enter a number: ');
if num > 0
    disp('The number is positive.');
elseif num < 0
    disp('The number is negative.');
else
    disp('The number is zero.');
end

üí° Tip: Use elseif to handle multiple conditions without having to nest multiple if statements.

üîÑ Analogy:

Think of an if-elseif-else statement as deciding what to wear based on the weather. If it‚Äôs raining (if), wear a raincoat. If it‚Äôs cold (elseif), wear a jacket. Otherwise (else), wear a T-shirt.

Switch Statement:

Definition: A switch statement allows you to select one of many code blocks to execute.

Syntax:

switch expression
    case value1
        % Code to execute if expression == value1
    case value2
        % Code to execute if expression == value2
    otherwise
        % Code to execute if expression does not match any case
end

Example:

day = input('Enter a day number (1-7): ');
switch day
    case 1
        disp('Sunday');
    case 2
        disp('Monday');
    case 3
        disp('Tuesday');
    case 4
        disp('Wednesday');
    case 5
        disp('Thursday');
    case 6
        disp('Friday');
    case 7
        disp('Saturday');
    otherwise
        disp('Invalid day number');
end

üí° Tip: Use switch statements for clean, easy-to-read code when dealing with multiple specific cases.

üîÑ Real-World Analogy:

A switch statement is like a vending machine. You press a button (expression) and it dispenses the corresponding item (case). If you press a button that doesn‚Äôt exist, it shows an error message (otherwise).

Comparative/Relational Operators:

How to define a condition

Operators: Used to compare values.

  • == : Equal to
  • ~= : Not equal to
  • < : Less than
  • > : Greater than
  • <= : Less than or equal to
  • >= : Greater than or equal to

Example:

a = 5;
b = 3;
if a > b
    disp('a is greater than b');
end

‚ö†Ô∏è Gotcha: Don‚Äôt confuse = (assignment) with == (equality). Using = in a condition can lead to unexpected results!

Logical Operators:

Operators: Used to combine multiple conditions.

  • | : OR
  • & : AND
  • ~ : NOT
  • || : Short-circuit OR (stops evaluating if the first condition is true)
  • && : Short-circuit AND (stops evaluating if the first condition is false)

Example:

x = true;
y = false;
if x & ~y
    disp('x is true and y is false');
end

üîÑ Real-World Analogy:

Logical operators are like combining multiple conditions in a checklist. For example, to decide to go outside, you might check if it‚Äôs sunny AND warm, or if it‚Äôs not raining.

Loops:

Loops allow you to repeat a block of code multiple times.

for Loop:

Definition: Repeats a block of code a fixed number of times.

Syntax:

for index = start_value
    % Code to execute in each iteration
end

Example:

for i = 1:5
    disp(['Iteration: ', num2str(i)]);
end

üí° Tip: Use for loops when you know in advance how many times you need to repeat a task.

üîÑ Real-World Analogy:

A for loop is like counting the number of laps you run around a track. You know you need to run 5 laps, so you repeat the same action 5 times.

while Loop:

Definition: Repeats a block of code while a condition is true.

Syntax:

while condition
    % Code to execute while condition is true
end

Example:

i = 1;
while i <= 5
    disp(['Iteration: ', num2str(i)]);
    i = i + 1;
end

‚ö†Ô∏è Gotcha: Be careful with while loops! Ensure the condition will eventually be false to avoid infinite loops.

üîÑ Real-World Analogy:

A while loop is like filling a glass of water. You keep pouring until the glass is full. You don‚Äôt know exactly how much water it will take, but you stop when the condition (full glass) is met.

Break and Continue:

break: Exits the loop immediately.

continue: Skips the rest of the code in the current iteration and proceeds to the next iteration.

Example (Break):

for i = 1:10
    if i == 5
        break;
    end
    disp(i);
end

Example (Continue):

for i = 1:10
    if mod(i, 2) == 0
        continue;
    end
    disp(i);
end

‚ö†Ô∏è Gotcha: Use break and continue carefully to avoid creating logic that's difficult to follow!

üîÑ Real-World Analogy:

break: Imagine you are searching for a specific book in a library. Once you find it, you stop searching and leave (exit the loop).

continue: Imagine you are checking a list of items for defects. If you find a defective item, you skip further inspection of that item and move to the next (skip iteration).

Input and Output üì§üì•

Input:

Static/Hard-coded Input:

Assign values directly using the equal sign.

Dynamic Input:

Use the input() function to get user input.

radius = input('Please enter the radius: ');

Example:

height0 = input('Please enter the initial height (in meters): ');
time = input('Please enter the time the object was in the air (in seconds): ');
velocity0 = input('Please enter the initial velocity (in m/s): ');
ACC = 9.81;

heightFinal = height0 + velocity0 * time + 0.5 * ACC * time^2;
disp(['The final height is ', num2str(heightFinal), ' meters.']);

üí° Tip: Always prompt users clearly so they know what type of input is expected.

Output:

Capturing Output:

Use the diary function to capture Command Window output.

diary 'output.txt'
disp('Hello, World!');
diary off

Formatting Output:

disp():

someVariable = 25;
disp('The value is: ');
disp(someVariable);

fprintf():

fprintf('The value is %d\n', someVariable);

sprintf():

str = sprintf('The value is %d', someVariable);
disp(str);

Example:

weight = 57638.3333;
fprintf('The weight is %8.2f pounds\n', weight);
fprintf('The weight is %50.2f pounds\n', weight);

String Input:

Inputting Strings: Use the input() function with the 's' argument for strings.

name = input('Please enter your name: ', 's');
disp(['Hello, ', name, '!']);

Example:

username = input('Please enter your name: ', 's');
fprintf('You said your name was %s, hello %s!\n', username, username);

Format Specifiers: used with fprintf and sprintf

  • %d or %i: Integer
  • %f: Floating-point number
  • %s: String
  • %g: Compact floating-point format
  • %5.2f: Floating-point number with 5 characters wide and 2 digits after the decimal point
  • \n: Newline character

Example:

fprintf('Integer: %d\n', 42);
fprintf('Floating-point: %8.2f\n', 3.14159);
fprintf('String: %s\n', 'Hello, world!');

Using sprintf:

str = sprintf('The value is %8.2f', 3.14159);
disp(str);

Example:

fileIncrement = 1;
fileName = sprintf("My file (%i)", fileIncrement);
disp(fileName); % Outputs: My file (1)
